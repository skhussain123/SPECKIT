
### 1. AI Assistant
An AI Assistant is a tool or agent that helps humans perform tasks — using AI models (like ChatGPT, Copilot, or Gemini).

#### Example : 
* ChatGPT answering code or research queries
* GitHub Copilot helping in code completion
* Google Assistant or Siri doing voice tasks

### 2. AI-Driven Development
AI-driven development means existing software systems or workflows me AI ka integration karna — taake wo smart ho jayein.
AI is a helper layer, not the foundation.Traditional software pe AI “add” kiya jata hai.

#### Example:
* Existing app me AI chatbot add karna
* CRM system me AI-based customer prediction lagana
* IDE me AI code suggestion add karna (Copilot, Tabnine)

### 3. AI-Native Development
AI-native development wo hota hai jahan AI is the core of the system, not just an add-on.
Yani, app ka main logic, workflow, aur architecture AI pe depend karta hai.System khud AI ke around bana hota hai — not AI added later.

#### Example :
* Existing app me AI chatbot add karna
* CRM system me AI-based customer prediction lagana
* IDE me AI code suggestion add karna (Copilot, Tabnine)
